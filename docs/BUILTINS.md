# BUILTINS
<!----------------------------------------------------------------------------->

So this is a kind of first sketch of the minimum functionality that we will 
support. It needs more work and more details. 

<!----------------------------------------------------------------------------->
## Boolean Operations
<!----------------------------------------------------------------------------->

### Unary Operators

`(Any) => Bool`

- not, !
- defined

### Binary Operators

`(Any, Any) => Bool`

- eq, ==
- ne, != 

- ge, >=
- gt, >
- le, <=
- lt, <

- and, &&
- or,  ||
- xor, //

`(Any, ClassName) => Bool`

- isa
- ref

`(Any, Any) => -1 | 0 | 1`

- cmp, <=>
